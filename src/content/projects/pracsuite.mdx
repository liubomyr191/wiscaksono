## PracSuite
PracSuite is a cloud-based practice management software for healthcare professionals. It offers a wide range of features, including appointment booking, clinical notes, patient records, online forms, and invoicing. PracSuite is designed to help healthcare professionals improve their efficiency and productivity, while also providing a better experience for their patients.

### Features
- Appointment booking: PracSuite's appointment book makes it easy to schedule appointments for patients, both in-person and online.",
- Clinical notes: PracSuite's clinical notes system makes it easy to create and store detailed notes about patient visits.",
- Patient records: PracSuite's patient records system provides a central repository for all of a patient's medical information.",
- Online forms: PracSuite's online forms make it easy to collect patient information and obtain consent.",
- Invoicing: PracSuite's invoicing system makes it easy to send invoices to patients and track payments."

### Stack
- [Next](https://nextjs.org) - Next.js is a React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
- [GSAP](https://greensock.com/gsap/) - GSAP is a robust JavaScript toolset that turns developers into animation superheroes
