## BigFriend
Bigfriend is a community of communication agencies and consulting firms that are dedicated to providing advice and support to businesses and individuals. The website provides information about Bigfriend's services, as well as a blog with articles on a variety of topics related to communication and marketing.


### Features
- Communication strategy: Bigfriend can help businesses develop and implement communication strategies that are aligned with their overall business goals.
- Content creation: Bigfriend can create content for businesses, including website content, blog posts, social media posts, and marketing materials.
- Public relations: Bigfriend can help businesses manage their public relations efforts, including media relations, crisis communications, and event planning.
- Digital marketing: Bigfriend can help businesses develop and implement digital marketing campaigns, including search engine optimization (SEO), pay-per-click (PPC) advertising, and social media marketing

### Stack
- [React](https://nextjs.org) - Next.js is a React framework for building full-stack web applications
- [Next](https://nextjs.org) - Next.js is a React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
- [Redux](https://redux.js.org) - Redux is an open-source JavaScript library for managing and centralizing application state.
