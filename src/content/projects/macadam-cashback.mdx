## Macadam Cashback: Redefining Online Savings
Discover the future of online shopping with Macadam Cashback, a Chrome extension designed to make earning cashback effortless and enjoyable. Developed using ReactJS and Tailwind CSS, this extension boasts a stunning 90% UI conversion from Figma, ensuring a visually pleasing and user-friendly interface for a seamless cashback experience.
[Download it here](https://chromewebstore.google.com/detail/macadam-cashback/olfglomegbapljcohegnfnkpmmdbeiln).

### Features
- **Intuitive UI Conversion:** Benefit from a visually appealing and user-friendly interface. 
- **Real-time Cashback Alerts:** Macadam Cashback seamlessly integrates with browsers, providing users with real-time cashback alerts as they shop online, adding excitement to the shopping experience.
- **Effortless Browser Integration:** The extension works across various e-commerce platforms, automatically detecting eligible purchases and rewarding users with cashback without any additional steps.

### Stack
- [React](https://react.dev) - Next.js is a React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
