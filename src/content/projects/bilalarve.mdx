## Bilal Arve's Personal Site
A personal website of Bilal Arve, a UI/UX designer based in Banda Aceh, Indonesia. The website is built with Next.js and TailwindCSS, and it uses Lottie animations to add some visual interest.
[Preview](https://www.bilalarve.com/)


### Features
- Animation: The website uses Lottie animations to add some visual interest.
- Responsive design: The website is responsive and works well on mobile devices.
- Accessibility: The website is accessible and works well with screen readers

### Stack
- [Next](https://nextjs.org) - Next.js is a React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
- [Lottie](https://lottiefiles.com/) - is a JSON-based animation file format that allows you to ship animations on any platform as easily as shipping static assets
