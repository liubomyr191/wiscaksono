## Agenone
is a website that allows users to mint NFTs (non-fungible tokens) that represent physical poker decks. Each NFT gives the user access to a private Discord server and regular whitelist spots for other crypto projects. The website also offers a spinwheel game where users can win real poker decks.
[Preview](https://xcards.gg/).

### Features
- NFTs: HODLCards.net allows users to mint NFTs that represent physical poker decks. These NFTs can be used to access a private Discord server and regular whitelist spots for other crypto projects.
- Spinwheel: HODLCards.net also offers a spinwheel game where users can win real poker decks. The spinwheel game is free to play and users can win a deck of cards with every spin.
- Community: HODLCards.net has a growing community of crypto enthusiasts who are passionate about poker. The community is active on Discord and users can connect with other players to participate in tournaments and discuss strategies.
